## 🤔 什么是 Gemini Omni？

**Gemini Omni** 是 **Google** 开发的视频生成模型，支持通过图片、视频、音频和文本等多模态输入生成视频。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐
